**Develocoeur** — *Création de sites web accessibles, humains et adaptés à votre budget.*
 
C'est l'agence web derrière ce dépôt : développement freelance centré sur la proximité client, la communication et le travail bien fait. Le site que vous consultez est à la fois la vitrine de l'agence et une démonstration concrète de ce qu'on sait faire.
 
## Ce qu'on propose
 
- **Création de sites web** — vitrine, portfolio, e-commerce, sur-mesure
- **Applications web** — full stack, du front au back
- **SEO** — référencement naturel et optimisation des performances
- **Maintenance & support** — suivi dans la durée, pas juste à la livraison
- **Formation** — accompagnement pour les équipes ou les indépendants
- **Et tout le reste** — si vous avez un besoin, on en parle !
> Vous avez un projet ? Rendez-vous sur [mathieuescabias.github.io](https://mathieuescabias.github.io/) 👋
 
## Stack technique
 
- [Astro](https://astro.build/) — framework principal
- [Tailwind CSS](https://tailwindcss.com/) — styles utilitaires
- [React](https://react.dev/) — composants interactifs
- [TypeScript](https://www.typescriptlang.org/) — typage statique
- [SCSS](https://sass-lang.com/) — styles avancés
- [MDX](https://mdxjs.com/) — contenu enrichi
## Fonctionnalités
 
### Contenu
 
- **Blog** — articles avec catégories, tags et temps de lecture estimé
- **Projets** — vitrine de mes réalisations
- **Page d'accueil**, page 404 et mentions légales
#### Composants de contenu
 
- **Blocs de code** avec coloration syntaxique
- **Support LaTeX** pour les expressions mathématiques inline et en bloc
- **Vidéos YouTube** intégrées
- **Accordéons** et sections à onglets
- **Blocs de notice** — note, astuce, info, avertissement
- Toutes les fonctionnalités **MD/MDX** standard (notes de bas de page, tableaux, etc.)
### Fonctionnalités additionnelles
 
- **Mode clair / Mode sombre** avec détection automatique des préférences système
- **Recherche** intégrée sur l'ensemble du contenu (via Pagefind)
- **Mise en page adaptative** — mobile, tablette, desktop
- **Info-bulles** sur tous les éléments interactables
- **Fil d'Ariane** pour naviguer rapidement
- **Contenu connexe** généré automatiquement
- **Temps de lecture** calculé automatiquement
- **Transitions animées** entre les pages
- **Pagination** sur toutes les collections de contenu
- **Partage sur les réseaux sociaux**
- **SEO** — sitemap, RSS et métadonnées structurées intégrés
- Thème de couleurs et typographie entièrement personnalisables
